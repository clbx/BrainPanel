Brain Panel: v0.1

This is a beta version of a panel to display Muse Brain wave information a realtime and post-record way

Real Time Goals:
  • Have a graph with real-time information from the muse brain sensor.
  • Receive real-time statistics from the muse.
  • Receive information on the device itself

Post-Record Goals:
  • Display out helpful information about the session
  • Display Statistics about session
  • Display a graph with the information from the session
